About the animation

Animating the parameters of a hillshade function to simulate the illumination of an elevation surface during a summer day. Red River Gorge, Kentucky.

Data used in this project

Elevation data was extracted from Kentucky's Elevation Data & Aerial Photography Program 5-foot resolution DEM data. A ArcGIS Python script was used to iterate through parameter values. ImageMagick was then used to assemble an animated GIF. The GIF is 49 frames at 2422 x 1758 pixels.
